# Obsidian Writer Skill

## 역할

분석된 YouTube 인사이트를 Obsidian 볼트의 규칙에 맞는 노트로 변환하여 저장합니다.

## 볼트 정보

- **볼트 경로**: `~/.claude/settings/youtube-vault-path.txt` 파일에서 읽기. 없으면 사용자에게 경로를 물어본 후 해당 파일에 저장
- **YouTube 노트 위치**: `30_Resource/10_YouTube/[카테고리]/`

## 카테고리 폴더 매핑

| 카테고리 | 폴더 경로 |
|----------|-----------|
| 자기계발 | `30_Resource/10_YouTube/01_자기계발/` |
| 투자 | `30_Resource/10_YouTube/02_투자/` |
| 커리어 | `30_Resource/10_YouTube/03_커리어/` |
| 기술 | `30_Resource/10_YouTube/04_기술/` |
| 비즈니스 | `30_Resource/10_YouTube/05_비즈니스/` |
| 기타 | `30_Resource/10_YouTube/99_기타/` |

## 파일명 규칙

```
YYMMDD_영상제목_요약.md
```

- **YYMMDD**: 노트 작성일 (예: 260208)
- **영상제목**: 영상 제목에서 핵심 키워드 추출 (20자 이내)
- **요약**: 핵심 내용 한 줄 요약 (10자 이내)
- 파일명에 사용 불가 문자 (`\/:*?"<>|`) 제거
- 공백은 언더스코어(`_`)로 대체

예시:
- `260208_아침루틴_5가지습관.md`
- `260208_AI에이전트_개발패턴.md`
- `260208_ETF투자_초보가이드.md`

## YAML Frontmatter 템플릿

```yaml
---
title: "영상 원본 제목"
source: "https://www.youtube.com/watch?v=VIDEO_ID"
channel: "채널명"
date: YYYY-MM-DD
tags:
  - youtube
  - 카테고리태그
  - 키워드1
  - 키워드2
  - 키워드3
language: ko
category: 카테고리명
---
```

### Frontmatter 규칙
- `title`: 영상 원본 제목 그대로 (따옴표 감싸기)
- `source`: 전체 YouTube URL
- `channel`: 채널명
- `date`: 노트 작성일 (YYYY-MM-DD)
- `tags`: 최소 3개, 최대 7개. 항상 `youtube` 태그 포함. 한국어 태그 사용
- `language`: 자막 언어 코드 (ko, en, ja 등)
- `category`: 분류된 카테고리명

## 노트 본문 구조

```markdown
# 영상 제목

## 📋 기본 정보

| 항목 | 내용 |
|------|------|
| 채널 | 채널명 |
| 링크 | [YouTube 영상](URL) |
| 언어 | 한국어/영어/일본어 |
| 카테고리 | [[카테고리]] |

## 📝 핵심 요약

(3-5문장 요약)

## 💡 주요 인사이트

(타임스탬프 포함 인사이트 목록)

## 📖 상세 노트

(구간별 상세 정리)

## 🎯 실전 적용 전략

(체크박스 액션 아이템)

## 💬 인상 깊은 인용구

(인용구 목록)

## 🔍 비판적 시각

(비판적 분석)

## 🏷️ 관련 노트

(관련될 수 있는 노트 위키링크)
```

## 포맷 규칙

### 위키링크
- 관련 개념이나 노트에 `[[위키링크]]` 사용
- 태그는 frontmatter에서만 관리 (본문에 `#태그` 사용 금지)

### 타임스탬프
- `[MM:SS]` 형식 사용
- 1시간 이상: `[H:MM:SS]` 형식

### 언어
- 분석 결과는 **한국어**로 작성
- 영어 영상이어도 분석 노트는 한국어
- 고유명사, 전문 용어는 원문 병기 가능: `에이전트(Agent)`

### 이모지
- 섹션 제목에만 이모지 사용 (위 구조 참조)
- 본문 내용에는 이모지 사용하지 않음

## 저장 프로세스

1. 카테고리에 맞는 폴더 경로 결정
2. 파일명 생성 (YYMMDD_제목_요약.md)
3. frontmatter + 본문 조합
4. Write 도구로 파일 저장
5. 저장된 파일의 절대 경로 반환
