# YouTube Insight 분석 Skill

## 역할

YouTube 자막 데이터를 분석하여 구조화된 인사이트를 생성하는 프레임워크입니다.
이 Skill은 `/youtube` 커맨드 파이프라인에서 자막 추출 후 자동으로 적용됩니다.

## 분석 프로세스

자막 데이터(JSON)를 입력받아 아래 7개 섹션을 **한국어**로 생성합니다.

### 1. 콘텐츠 분류

아래 카테고리 중 **하나**를 선택합니다:

| 카테고리 | 폴더 | 키워드 예시 |
|----------|------|-------------|
| 자기계발 | 01_자기계발 | 습관, 마인드셋, 생산성, 루틴, 독서, 성장 |
| 투자 | 02_투자 | 주식, 부동산, ETF, 경제, 재테크, 포트폴리오 |
| 커리어 | 03_커리어 | 이직, 면접, 연봉, 직장, 프리랜서, 사이드잡 |
| 기술 | 04_기술 | 프로그래밍, AI, 개발, 디자인, 툴, 자동화 |
| 비즈니스 | 05_비즈니스 | 창업, 마케팅, 브랜딩, 매출, 사업, 전략 |
| 기타 | 99_기타 | 위 카테고리에 해당하지 않는 경우 |

### 2. 핵심 요약

- **3-5문장**으로 영상의 핵심 메시지를 요약
- 첫 문장: 영상이 다루는 주제/문제
- 중간: 핵심 주장이나 해결책
- 마지막: 결론 또는 핵심 메시지

### 3. 주요 인사이트

- **3-7개** 인사이트를 추출
- 각 인사이트에 **타임스탬프** [MM:SS] 포함
- 형식: `- [MM:SS] **인사이트 제목**: 상세 설명`
- 가장 중요하고 실행 가능한 내용을 우선 배치

### 4. 상세 노트

- 영상을 **논리적 구간**(3-6개)으로 나누어 정리
- 각 구간에 타임스탬프 범위 표시: `### [00:00-05:30] 구간 제목`
- 구간 내용을 불릿 포인트로 정리
- 핵심 개념, 예시, 데이터를 포함

### 5. 실전 적용 전략 (Action Items)

- **구체적이고 실행 가능한** 행동 목록
- 체크박스 형식: `- [ ] 액션 아이템`
- 각 항목에 **기한 또는 조건** 포함 (예: "이번 주 내", "다음 회의에서")
- 우선순위 순으로 배치

### 6. 인상 깊은 인용구

- 영상에서 **직접 인용**할 만한 발언 2-4개
- 형식: `> "인용구 내용" — 화자명 [MM:SS]`
- 핵심 메시지를 잘 요약하는 문장 선택

### 7. 비판적 시각

- 영상 주장의 **한계점** 또는 **전제 조건**
- 누락된 관점이나 반론
- 적용 시 주의할 점
- 추가로 찾아볼 만한 참고 자료

## 타임스탬프 변환 규칙

자막 JSON의 `start` 값(초)을 [MM:SS] 형식으로 변환합니다:
- 60초 → [01:00]
- 125초 → [02:05]
- 3661초 → [1:01:01]

## 품질 기준

- 자막이 자동 생성(auto)인 경우, 노트 상단에 "⚠️ 자동 생성 자막 기반으로 일부 오류가 있을 수 있습니다" 경고 추가
- truncated 플래그가 true인 경우, "⚠️ 긴 영상으로 자막이 잘렸습니다. 후반부 내용이 누락될 수 있습니다" 경고 추가
- 자막 품질이 낮아 의미 파악이 어려운 구간은 "[불명확]"으로 표시
